== Description ==

Repeat Events | Simple Event Listing WordPress Plugin | SEO Optimised

Single Events
- Create single events on a specific date (eg. 25th December)

Repeating Events
Select the days on which the event occurs (eg. Every Friday and Saturday)

Event Listing Page
The events will be displayed in a simple chronological order, incorporating single, series and repeating.

SEO optimised
Where possible, the plugin will use the same permalink for events. This not only will improve the SEO performance of your events but it will also mean the content of your event can be edited in once place.

ACF Fields
Fields are created using ACF (Advanced Custom Fields) so can be easily extended if required.

Template Overrides
The plugin have default archive and single pages. If you require to override these. You can create the following files in your theme:
repeat-events/single-bluebox_repeat_event.php
repeat-events/archive-bluebox_repeat_event.php
